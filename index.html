Computational Soft Robot Exploration


WOW, my capstone project LOLüò≤. The decision of this project is made because there is almost no existing finite element analysis(FEA) solver with the ability to model high deformation thin paper-based origami chambers. This big project consists of mechanical thin shell and rigid body design, electrical DC motor controller design, PID control system applied on the pressure control station and an modified extension package of ArcSim, a thin shell FEA simulation software.
